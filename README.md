# Komanda

**Komanda** (codename: Kilo) is part of the [Strategos](https://strategos.au)
Suite — professional forensic investigation platform.

Komanda provides case management and case-level chain of custody across
the Strategos Suite.

This package is a namespace holder. Installing `komanda` automatically
installs `komanda-kilo`, the active development package.

## Install

pip install komanda-kilo
